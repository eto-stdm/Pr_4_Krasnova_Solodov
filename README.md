### Описание:
Данный README-файл написан для проекта 4ой практической работы "ТЕСТИРОВАНИЕ БЕЛЫМ
ЯЩИКОМ Часть 1" по дисциплине "Поддержка и тестирование программных модулей".

Цель работы: приобрести практические навыки ручного тестирования методом "белого ящика".
Разработчики: Краснова А.А. и Солодов А.Е.
Учебная группа: 3ИСИП-123
Вариант задания: 1

### Используемый стек технологий:
- Приложение WPF (.NET Framework)
- C#

### Описание архитектуры:
Фронтенд приложения построен на базе WPF (.NET Framework), а бэкенд на класическом C#.

### Основные компоненты
- 3 объекта Page.xaml (и Page.xaml.cs) с 3 разными заданиями.

========== Page1: ==========
- Объект Image с формулой
- 4 TextBlock с информацией о полях 4-ех TextBox
- Кнопка(Button) для вычисления
- Кнопка(Button) для очищения полей
- Кнопка(Button) для перехода на следующую страницу
- Функция CalculateBtn_Click, которая проверяет вводимые данные
- Функция Calculate, которая высчитывает по самой формуле

========== Page2: ==========
- Объект Image с формулой
- Объект Image с котиком
- Объект Image c фразой котика
- 3 TextBlock с информацией о полях 3-ех TextBox
- 3 RadioButton для выбора типов функции
- Кнопка(Button) для вычисления
- Кнопка(Button) для очищения полей
- Кнопка(Button) для перехода на следующую страницу
- Кнопка(Button) для перехода на предыдущую страницу
- Функция для проверки данных и вычисления по формуле
- Функция замедленного исчезновения фразы котика

========== Page3: ==========
- Объект Image с формулой
- 3 TextBlock с информацией о полях 3-ех TextBox
- Объект WindowsFormsHost для вывода графика
- Кнопка(Button) для вычисления
- Кнопка(Button) для очищения полей
- Кнопка(Button) для перехода на предыдущую страницу
- Функция CalculateBtn_Click, которая проверяет вводимые данные
- Функция Calculate, которая высчитывает по самой формуле

### Взаимодействие:
Пользователь вводит значения в поля TextBox, после чего может нажать на кнопку
"Вычислить" --> выведет ответ или "Очистить" --> поля ввода и вывода очистятся.
Также присутсвует взаимодействие с кнопками "Далее" и "Назад", которые выполняют
соответсвующие для них функции. При закрытии вылезает предупреждение о закрытии
программы.
